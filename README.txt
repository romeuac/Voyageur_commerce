1 - En utilisant le Qt faire la compilation du projet "tsp" qui est dans le dossier "tsp"

2 - Il faut mettre dans le dossier crée par la compilation les fichiers qui sont dans les fichiers .rar :

branco.png

france.png

intro.jpg

villes.txt

westernSaara.tsp

3 - Exécuter le fichier "tsp" (le compilé)

4 - Appuyer sur le bouton "Load" pour charger le fichier de "villes.txt" (Par exemple).

5 - Choisir les autres paramètres comme vous voulez (taille de la population, croisement et mutation).

6 - Appuyer sur le bouton "Trouver chemin" et attendre la réponse.
